xLights Moving Head .xmodel Builder

Generate a DMX moving-head .xmodel for any quantity of heads (up to 8).

Requires xLights 2026.09 or newer — earlier versions don't support multi-model .xmodel import.

Color Properties

Pick how this Moving Head produces color. Color Wheel = a rotating wheel with fixed color slots (one DMX channel). RGBW = separate Red, Green, Blue (and optional White) LED channels.

DMX value is where the wheel lands on that color. Single number (4) or range (2-5).

Welcome to the xLights Moving Head .xmodel Builder

    

      
This tool generates ready-to-import .xmodel files for xLights, for any DMX moving-head fixture.

      
⚠️ Requires xLights 2026.09 or newer — earlier versions don't support multi-model .xmodel import.

      
Quick start:

      
    
        
  1. Pick your fixture from the Choose preset dropdown (or switch to Add new Moving Head if it isn't listed).
    2. 
        
  2. Set the Quantity of heads for your show.
    3. 
        
  3. Pick the Head orientation at rest.
    4. 
        
  4. Click Generate and Download xmodel.
    5. 
        
  5. In xLights Layout, click the Import icon, drag a box on the layout, and choose your .xmodel file from your Downloads folder.
    6. 
      

      
For full details, the channel-type reference, and how to submit a new fixture to the catalog, see the README on GitHub.
